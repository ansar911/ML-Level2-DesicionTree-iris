## 📌 Project Overview

This project focuses on building a Decision Tree classifier to predict the species of flowers using the Iris dataset.

---

## 🧠 Objective

* Train a classification model
* Understand how decision trees split data
* Evaluate model performance

---

## 📊 Dataset

* Iris Dataset
* Features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
* Target:

  * Species (setosa, versicolor, virginica)

---

## 🤖 Model Used

* Decision Tree Classifier (Scikit-learn)

---

## 📈 Model Evaluation

* Accuracy Score
* Precision
* Recall
* F1-Score

### ✅ Result

* Achieved **~93% accuracy**
* Model performed well across all classes

---

## 🌳 Visualization

* Visualized decision tree structure to understand decision-making process

---

## 🧠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

---

## 🔗 Future Improvements
